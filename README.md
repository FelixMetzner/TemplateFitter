# TemplateFitter

## Installation

Get the repository via
```
git clone https://github.com/welschma/TemplateFitter
```
or 
```
git clone git@github.com:welschma/TemplateFitter.git
```
and install the package with
```
cd TemplateFitter
pip3 install -r requirements.txt
pip3 install -e .
```
